Tetris

This web application was created using React. It is used to play Tetris and has three modes.
1.single player mode
2.multiplayer mode
3.play with friend mode

In single player mode, you can play the classic Tetris

InMultiplayer mode, you will be matched with an opponent to battle it out.
The player with the higher score will emerge victorious.
After the match, you can either propose a rematch or exit the multiplayer mode.
